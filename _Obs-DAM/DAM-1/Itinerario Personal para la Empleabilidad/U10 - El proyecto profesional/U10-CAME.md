# 🧭 Matriz CAME

La **matriz CAME** es una herramienta complementaria al DAFO que permite **transformar el análisis en un plan de acción**.

Se construye a partir de la matriz DAFO.

## Significado de CAME

- **Corregir** las debilidades
- **Afrontar** las amenazas
- **Mantener** las fortalezas
- **Explotar** las oportunidades

La matriz CAME permite:
- Definir estrategias personales
- Establecer acciones concretas
- Orientar el proyecto profesional hacia el éxito

> [!tip]
> DAFO analiza → CAME actúa.
