# 🧾 Introducción a XML

## XML y lenguajes de marcas

- Un **lenguaje de marcas** usa etiquetas para describir y estructurar información.
- Las etiquetas **no son el contenido**, indican qué representa ese contenido.
- XML es un lenguaje orientado a **almacenar e intercambiar datos estructurados**.
- A diferencia de HTML, **XML no está pensado para mostrar**, sino para describir datos.

---

## ¿Qué es XML y para qué sirve?

- XML significa **eXtensible Markup Language**.
- Permite representar información de forma estructurada.
- Se usa para:
  - intercambio de datos entre aplicaciones
  - archivos de configuración
  - catálogos y documentos estructurados

---

## Ejemplos de uso

### Intercambio de datos
