## 1. Introducción a CSS

El lenguaje **CSS** (Cascading Style Sheets) fue diseñado para separar el contenido (HTML) de la parte estética y visual de una web. Mientras que el HTML define **qué** debe visualizarse, el CSS define **cómo** debe visualizarse.

### Sintaxis Básica

Una regla de CSS se compone de un **selector** y un **bloque de declaración**.

- **Selector**: Indica a qué elemento(s) HTML afecta la regla (ej. `h1`, `.clase`, `#id`).
    
    
- **Declaración**: Se escribe entre llaves `{}` y contiene una o más parejas de **propiedad: valor**, separadas por punto y coma.
    
    

### Ubicación y Prioridad

Existen tres formas de integrar CSS, con distintos niveles de prioridad (de mayor a menor):


1. **CSS en línea**: Atributo `style` dentro de la etiqueta HTML.
        
2. **CSS interno**: Etiqueta `<style>` dentro del `<head>` del documento.
        
3. **CSS externo**: Archivo independiente `.css` vinculado mediante `<link>` (Recomendado para reutilización).
