## ¿Para qué sirve la monitorización?

> [!info] Objetivo
> Ver el **tráfico** que circula por la red para:
> - Comprobar que funciona correctamente
> - Detectar **sobrecargas** o **fallos**
> - Identificar posibles **ataques o intrusos**

Si la red empieza a fallar o funcionar lento → monitorizar con alguna herramienta.

---

## IPTraf-ng

- Solo **Linux**, basado en línea de comandos
- Licencia: **GNU GPL v2**

```bash
sudo apt install iptraf-ng
sudo iptraf-ng        # Requiere permisos de administrador
```

### Funciones disponibles
- Monitorizar el tráfico IP de la red
- Estadísticas de las interfaces
- Estadísticas de fallos
- Monitor de la red LAN
- Filtros y configuración

---

## Nmon

- Solo **Linux**
- Licencia: **GPLv3**
- Monitoriza tanto red como sistema completo

```bash
sudo apt install nmon
nmon
```

### Teclas interactivas

| Tecla | Función |
|-------|---------|
| `n` | Monitorizar **red** |
| `c` | Monitorizar CPU |
| `m` | Monitorizar memoria |
| `d` | Monitorizar discos |
| `k` | Monitorizar kernel |
| `V` | Monitorizar memoria virtual |
| `t` | Monitorizar procesos |
| `q` | Salir |

## WireShark

> [!info] WireShark
> Analizador de **tráfico de red** muy completo y con interfaz gráfica.
> Permite capturar y analizar paquetes en tiempo real.

- Multiplataforma (Linux, Windows, macOS)
- Web: [wireshark.org](https://www.wireshark.org/)

---

## Kismet

> [!info] Kismet
> Analizador especializado en **redes inalámbricas**.
> Detecta redes WiFi, dispositivos y tráfico inalámbrico.

- Web: [kismetwireless.net](https://www.kismetwireless.net/)

## Advanced IP Scanner

> [!info] Advanced IP Scanner
> Escanea y analiza todos los **dispositivos de una red LAN**.

- Muestra: IPs, MACs, carpetas compartidas, servidores
- Disponible en versión portable (sin instalación)
- Web: [advanced-ip-scanner.com/es](https://www.advanced-ip-scanner.com/es/)

## Nmap

> [!info] Nmap — Network Mapper
> Software libre (**GNU GPLv2**) para análisis de redes.
> Interfaz gráfica: **Zenmap**

```bash
sudo apt install nmap
# o
sudo snap install nmap
```

- Web: [nmap.org](https://nmap.org/)

### Sintaxis

```bash
nmap [tipo_sondeo] [opciones] objetivo
```

El **objetivo** puede ser: nombre de equipo, IP, rango de IPs o red completa.

### Ejemplos de uso

```bash
# Escanear una IP concreta
nmap 192.168.0.32

# Sondear un puerto específico
nmap -p 80 localhost

# Detectar SO y servicios
nmap -A -v equipo.dominio.edu

# Listar equipos en una red
nmap -sL -v 192.168.0.0/24

# Escanear todos los equipos de una red
nmap -sn -vv 192.168.1.*

# Guardar resultado en fichero
nmap -A 127.0.0.1 -oN fichero.txt
```

### Opciones útiles

| Opción | Descripción |
|--------|-------------|
| `-v` | Verbose: más detalles en la salida |
| `-vv` | Más detalles aún |
| `-p <puerto>` | Sondear un puerto específico |
| `-A` | Detección de SO, servicios y versiones |
| `-sL` | Listar equipos sin escanear |
| `-sn` | Ping scan: detectar equipos activos sin sondear puertos |
| `-oN <fichero>` | Guardar resultado en fichero de texto |

> [!warning] Importante
> Solo se debe usar Nmap en redes sobre las que se tiene **permiso** para realizar el análisis.

## ¿Para qué sirve un simulador?

> [!tip] Ventajas de simular antes de instalar
> - Detectar y corregir **errores de diseño** sin coste de hardware
> - Probar configuraciones antes del despliegue real
> - Herramienta de **aprendizaje** sin necesidad de hardware físico

---

## Cisco Packet Tracer

> [!info] Cisco Packet Tracer
> Simulador de redes desarrollado por **Cisco**.
> Muy utilizado en entornos **educativos** (certificaciones CCNA).

- Permite simular routers, switches, PCs, servidores...
- Incluye modo de simulación paso a paso del flujo de paquetes
- Gratuito para uso educativo (requiere cuenta Cisco NetAcad)

---

## GNS3 — Graphical Network Simulator 3

> [!info] GNS3
> Simulador de redes **libre y de código abierto**.
> Más cercano a equipos reales que Packet Tracer.

- Puede emular imágenes reales de IOS de Cisco
- Integra con **Dynamips** para emular routers Cisco
- Útil para configuraciones avanzadas y laboratorios profesionales
- Web: [gns3.com](https://www.gns3.com/)

---

## Dynamips

> [!info] Dynamips
> Emulador de **routers Cisco** de software libre.
> Sirve de base para GNS3.

- Permite ejecutar imágenes reales del sistema operativo IOS de Cisco

---

## Comparativa de simuladores

| Herramienta | Tipo | Nivel | Uso principal |
|-------------|------|-------|---------------|
| **Cisco Packet Tracer** | Simulador | Básico-Medio | Educación, certificaciones Cisco |
| **GNS3** | Emulador/Simulador | Medio-Avanzado | Laboratorios profesionales |
| **Dynamips** | Emulador | Avanzado | Base de GNS3, routers Cisco reales |
