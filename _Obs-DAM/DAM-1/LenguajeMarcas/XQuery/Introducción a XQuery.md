# 🧾 Introducción a XQuery

## ¿Qué es XQuery?

- XQuery (XML Query Language) es un lenguaje diseñado específicamente para **consultar y extraer datos de documentos XML**.
- Es un lenguaje funcional y declarativo.
- Está basado en **XPath**, extendiéndolo para permitir la creación de nuevos documentos XML.
- Es el equivalente a lo que es SQL para las bases de datos relacionales.

---

## Objetivos de XQuery

- Extraer datos específicos de uno o varios archivos XML.
- Transformar datos XML en otros formatos (como HTML, JSON o nuevo XML).
- Realizar cálculos y filtrados complejos sobre estructuras jerárquicas.

---

## Relación con XPath y XSLT

- **XPath**: Proporciona la sintaxis para navegar por el árbol XML (las "rutas"). XQuery utiliza XPath intensamente.
- **XSLT**: Se enfoca en la transformación de documentos. XQuery es más potente para la extracción y consulta de datos.
