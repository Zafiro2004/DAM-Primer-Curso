# 💪 Autoestima

  

La **autoestima** es la valoración que una persona hace de sí misma.

  

Se construye a partir de:

- El autoconocimiento

- La imagen personal

- La gestión de emociones

- Las experiencias vitales

  

Influye directamente en:

- La seguridad personal

- La toma de decisiones

- La gestión del estrés

- El afrontamiento de procesos de selección

  

> [!info]
> La autoestima **no es fija**: cambia a lo largo de la vida.
