## 5. Flexbox: El Diseño Flexible

Flexbox permite crear conjuntos de elementos que se adaptan automáticamente al contenedor y al tamaño de pantalla.

### Propiedades del Contenedor (Padre)

Para activar Flexbox, usamos `display: flex` o `inline-flex`.

- **Ejes**: Tiene un **eje principal** (por defecto horizontal) y un **eje secundario** (perpendicular al principal).
    
         
    
- **`flex-direction`**: Define la dirección (ej. `row` para horizontal, `column` para vertical).
    
        
    
- **`justify-content`**: Alinea los ítems en el **eje principal** (valores: `center`, `space-between`, `space-around`, etc.).
    
        
    
- **`align-items`**: Alinea los ítems en el **eje secundario** (valores: `stretch`, `flex-start`, `center`, etc.).
    
        
    
- **`flex-wrap`**: Permite que los elementos salten a la siguiente línea si no caben (`wrap`) o se mantengan en una sola (`nowrap`).
    
        
    

### Propiedades de los Ítems (Hijos)

- **`flex-grow`**: Capacidad de un ítem para crecer si hay espacio.
    
        
    
- **`flex-shrink`**: Capacidad de encogerse si no hay espacio.
    
        
    
- **`flex-basis`**: Tamaño inicial del elemento antes de crecer o encogerse.
    
        
    
- **`order`**: Cambia el orden visual de los elementos sin alterar el HTML.
    
        
    
- **`align-self`**: Permite a un ítem individual ignorar la alineación general del contenedor.
