La programación

Unidad 1: Introducción a la programación

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora cómo realizar una tarea. La programación puede realizarse utilizando varios lenguajes de programación, como Python, Java, C++, entre otros.

Java

Java es un lenguaje de programación de propósito general, concurrente, orientado a objetos y basado en clases. Fue diseñado para tener la menor cantidad de dependencias de implementación posibles, lo que permite que las aplicaciones escritas en Java puedan ejecutarse en cualquier sistema que tenga una máquina virtual de Java (JVM).

  • Python fue creado por Guido van Rossum y lanzado por primera vez en 1991.
  • Java fue desarrollado por James Gosling en Sun Microsystems y lanzado en 1995.
