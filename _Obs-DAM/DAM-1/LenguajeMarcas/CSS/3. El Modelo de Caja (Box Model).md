## 3. El Modelo de Caja (Box Model)

Todos los elementos CSS se estructuran como cajas con las siguientes capas (de afuera hacia adentro):

1. **Margin**: Espacio exterior fuera del borde.
    
2. **Border**: Línea que rodea el padding y el contenido. Puede tener estilos como `solid`, `dotted` o `dashed`.    
    
3. **Padding**: Espacio entre el contenido y el borde.
    
4. **Content**: El contenido real (texto, imagen, etc.).
