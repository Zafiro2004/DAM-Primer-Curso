## Por qué monitorizar y simular

- **Antes del montaje** → simular para detectar errores de diseño y ahorrar costes
- **Después del montaje** → monitorizar para detectar anomalías, sobrecargas o intrusiones

---

## Herramientas de Monitorización

### IPTraf-ng
- Solo **Linux**, basado en línea de comandos
- Licencia: **GNU GPL v2**
- Funciones: tráfico de red, estadísticas de interfaces, monitor LAN

```bash
sudo apt install iptraf-ng
sudo iptraf-ng
```

### Nmon
- Solo **Linux**
- Licencia: **GPLv3**
- Monitoriza: red, CPU, memoria, discos, kernel, procesos

```bash
sudo apt install nmon
nmon
# Pulsar 'n' para red, 'c' para CPU, 'm' para memoria, 'q' para salir
```

### WireShark
- Analiza el **tráfico de red** en detalle
- Web: [wireshark.org](https://www.wireshark.org/)

### Kismet
- Analizador para **redes inalámbricas**
- Web: [kismetwireless.net](https://www.kismetwireless.net/)

### Advanced IP Scanner
- Escanea **dispositivos en una red LAN**
- Muestra: IPs, MACs, carpetas compartidas, servidores
- Versión portable disponible (sin instalación)
- Web: [advanced-ip-scanner.com](https://www.advanced-ip-scanner.com/es/)

### Nmap
- Software libre, licencia **GNU GPLv2**
- Analiza: redes, estado de hosts, puertos abiertos, SO instalados
- Interfaz gráfica: **Zenmap**
- Web: [nmap.org](https://nmap.org/)

```bash
sudo apt install nmap

# Ejemplos de uso
nmap 192.168.0.32               # Escanear una IP
nmap -p 80 localhost            # Sondear puerto 80
nmap -A -v equipo.dominio.edu   # SO y servicios
nmap -sL -v 192.168.0.0/24      # Listar equipos en la red
nmap -sn -vv 192.168.1.*        # Listar y escanear equipos
nmap -A 127.0.0.1 -oN salida.txt # Guardar resultado en fichero
```

> `-v` = verbose (más detalles) · `-vv` = más detalles aún

## Simuladores de Red

> [!info] ¿Para qué sirve un simulador?
> Permite montar, probar y configurar una red **antes de su instalación real**, detectando errores sin coste de hardware.

| Simulador | Descripción |
|-----------|-------------|
| **Cisco Packet Tracer** | Muy usado en entornos educativos (Cisco) |
| **GNS3** | Software libre, más cercano a equipos reales |
| **Dynamips** | Emula routers Cisco, base de GNS3 |
