## Definiciones

> [!info] Protocolo
> Conjunto de **reglas o normas** que describen cómo se rigen las comunicaciones entre dispositivos de una red.

> [!info] Estándar
> **Formalización de un protocolo**, normalizado para que todos los que lo sigan puedan comunicarse entre sí.
> Son conceptos distintos pero relacionados.

### Funciones que regula un estándar
- Establecer el enlace
- Gestionar la transmisión
- Definir el acceso al medio
- Corregir y detectar errores

## Estándares Principales

### IEEE 802.3 — Ethernet
- Estándar para redes **LAN cableadas**
- Variantes: Fast Ethernet, Gigabit Ethernet, 100 Gigabit Ethernet
- Control de acceso al medio: **CSMA/CD**
- Con switches y conexiones dúplex se resuelve el problema del acceso al medio

### IEEE 802.11 — WiFi
- Estándar para **redes inalámbricas**
- Control de acceso al medio: **CSMA/CA** (evita colisiones)

### IEEE 802.15 — WPAN
- Redes inalámbricas de **área personal**
- **802.15.1** → Bluetooth
- **802.15.4** → ZigBee (domótica, ondas de radio de baja frecuencia)

### EIA/TIA T568A y T568B
- Normas para **cables de redes LAN**

### Protocolos en desuso
- **IEEE 802.5** → Token Ring (paso de testigo en anillo)
- **IEEE 802.4** → Token Bus (paso de testigo en bus)

## Historia: Protocolo Aloha

```
ALOHAnet (Universidad de Hawái)
    → Considerada base del estándar Ethernet
    → Usaba medios inalámbricos ("cable éter")
    → No verificaba si el canal estaba libre
        ↓
Aloha Ranurado
    → Divide el canal en tiempos para evitar colisiones
        ↓
CSMA
    → La estación escucha antes de transmitir
        ↓
CSMA/CD → usado en LAN (reducido con la llegada de switches)
CSMA/CA → usado en redes WiFi (evita la colisión)
```
