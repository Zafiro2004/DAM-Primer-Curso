# 🛠 Competencias profesionales

  

Las **competencias profesionales** son el conjunto de **habilidades y conocimientos técnicos** necesarios para desempeñar eficazmente una actividad laboral concreta.

  

Se adquieren principalmente mediante:

- Estudios reglados (FP, universidad)

- Formación complementaria

- Experiencia laboral

  

Son esenciales para:

- Realizar correctamente las tareas del puesto

- Progresar en una profesión específica

- Adaptarse a los cambios tecnológicos

  

## Tipos de competencias en FP

- **Competencia general del título**

- **Competencias profesionales específicas**

- **Competencias para la empleabilidad**
