# 🚀 Inserción profesional

  

La **inserción profesional** es el proceso mediante el cual una persona accede y se integra en el mercado laboral.

  

Depende de:

- La formación recibida

- Las competencias profesionales

- Las competencias personales y sociales

- El conocimiento del mercado laboral

- La autoestima y la actitud personal

  

Una buena inserción profesional requiere planificación y autoconocimiento.
