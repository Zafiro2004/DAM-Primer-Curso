## 2. Selectores y Atributos Globales

Los selectores permiten apuntar con precisión a los elementos que queremos estilizar.

|**Tipo de Selector**|**Sintaxis**|**Descripción**|
|---|---|---|
|**Universal**|`* { ... }`|Selecciona todos los elementos de la página.<br><br>|
|**Etiqueta**|`p { ... }`|Selecciona todos los elementos de ese tipo (ej. todos los párrafos).|
|**Clase**|`.nombre { ... }`|Se aplica a elementos con el atributo `class`. Puede usarse en varios elementos.|
|**ID**|`#nombre { ... }`|Se aplica a un único elemento con ese `id`. Debe ser único en el documento.<br><br> |
|**Descendiente**|`div p { ... }`|Selecciona elementos que están dentro de otros.|
|**Hijo directo**|`p > a { ... }`|Selecciona solo el primer nivel de descendencia.<br><br> |

**Nota:** Se pueden usar **pseudoclases** para definir estilos basados en el comportamiento del usuario, como `:hover` (ratón encima) o `:focus` (elemento activo).
