# Modificación sustancial de las condiciones de trabajo (MSCT)

  

Afecta a elementos esenciales del contrato.

  

## Elementos afectados

- Horario

- Jornada

- Sistema de turnos

- Salario

- Funciones

- Sistema de rendimiento

  

## Derechos del trabajador

- Aceptar

- Impugnar

- Extinguir el contrato → indemnización: **20 días/año** (máx. 12 meses)
