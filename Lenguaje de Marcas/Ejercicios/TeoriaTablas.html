Tablas en HTML

Las tablas se utilizan para organizar información en filas y columnas. La estructura básica incluye las etiquetas:

  • <table> define la tabla.
  • <tr> define una fila (table row).
  • <th> define una celda de encabezado (table header).
  • <td> define una celda de datos (table data).

Ejemplo básico de tabla

Una tabla sencilla con tres filas y tres columnas:

Nombre Edad Pais
Fernando Alonso 44 España
Lewis Hamilton 40 Inglaterra
Carlos 31 España

Observa que las etiquetas <th> aparecen en la primera fila y muestran el texto en negrita y centrado por defecto.

Partes de una tabla

Una tabla puede dividirse en tres partes para organizar mejor la información:

  • <thead>: cabecera
  • <tbody>: cuerpo
  • <tfoot>: pie
Ejemplo con cabecera y pie
Producto Precio Cantidad
Manzanas 2 € 10
Peras 1,5 € 8
Total 18

Nota: el atributo colspan="2" hace que esa celda ocupe dos columnas, útil para pies o títulos.

Hora Actividad
10:00 - 10:30 Recreo
10:30 - 11:00

Nota: el atributo rowspan="2" hace que esa celda ocupe dos filas

Tabla dentro de un párrafo explicativo

También se pueden añadir estilos más adelante con CSS, pero por ahora el atributo border="1" nos sirve para ver claramente las líneas.

Las tablas no deben usarse para maquetar la página (poner cosas en columnas), sino para mostrar datos estructurados.
