# 🔍 Análisis del potencial profesional

El análisis del potencial profesional consiste en **evaluar las capacidades personales y profesionales** para orientar la carrera laboral.

Incluye analizar:
- Intereses y motivaciones
- Habilidades y destrezas
- Cualidades personales
- Competencias profesionales
- Competencias personales y sociales
- Itinerarios formativos disponibles
- Mercado laboral

Este análisis permite formular:
- Objetivos profesionales adecuados
- Expectativas realistas
- Un plan de acción coherente
